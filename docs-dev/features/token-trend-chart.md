# Token 趋势曲线需求

## 目标
在排行榜页面展示前 5 用户的 token 消耗曲线，帮助观察趋势变化。

## 范围
- 支持窗口：7d / 14d / 30d / 90d
- 前 5 用户口径：按当前榜单（Tool + Ranking）前 5
- 曲线数据来源：snapshot 预计算的 `timeseries`

## 验收
1. 切换 Tool/Ranking，曲线前 5 与榜单前 5 一致。
2. 切换窗口，只改变时间范围和曲线，不改变前 5 选人。
3. 无数据时显示 `No trend data`，页面不报错。
4. 移动端可读，图例自动换行。

## 补充验收（tooltip + 回归）
5. 鼠标悬浮曲线区域时显示当日 tooltip（日期 + 用户值）。
6. `before` 与 `after` 截图必须按命名规范落盘，且可通过回归脚本校验。

## 表格字段调整（2026-03-04）
排行榜表格展示调整为：
- 主要工具 / 总Token数
- 输入/输出Token数
- 更新时间

## UI 增强（2026-03-04）
- 新增指标口径说明条（primary tool、token 列、更新时间口径）。
- 更新时间改为相对时间显示（hover 查看 UTC 原始时间）。
- 趋势图图例支持点击显隐曲线。
